# Line Height Styles

| Class | Properties |
| ----- | ---------- |
| lh-0 | line-height: 0 |
| lh-1 | line-height: 1 |
| lh-2 | line-height: 2 |
| lh-3 | line-height: 3 |
| lh-initial | line-height: initial |

You can achieve decimal values for line heights within the following range:
* 0.1 - 2.9
* Example = the `lh-2-1` class name will produce `line-height: 2.1`
